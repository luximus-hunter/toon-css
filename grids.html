Usage

Creating a Grid

To create a grid, use the grid class on a container element. You can specify the number of columns using classes like cols-1, cols-2, cols-3, etc. all the way up to cols-12.

<div class="grid cols-3 gap-md">
  <div class="card sm no-title">...</div>
  <div class="card sm no-title">...</div>
  <div class="card sm no-title">...</div>
</div>

Automatic Columns

If you want the grid to automatically adjust the number of columns based on the available space, you can use auto-sm up to auto-lg. This will create a responsive grid that adapts to different screen sizes.

<div class="grid auto-sm gap-md">
  <div class="card sm no-title">...</div>
  <div class="card sm no-title">...</div>
  <div class="card sm no-title">...</div>
  <div class="card sm no-title">...</div>
  <div class="card sm no-title">...</div>
</div>
