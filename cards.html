Card Examples

Toon CSS provides a versatile card component that can be used for various purposes, such as notifications, modals, and content containers. Below are some examples of different card styles and usages.

Usage

Basic Card

To create a card, simply add the card class to a container element. You can also add size classes like md to adjust the card's width.

Supported sizes: xxs, xs, sm, md, lg, xl and xxl.

Card Title

Use a h1 for the card title, and regular text elements for the content. h2 can be used for subheadings.

To create a card without a title, simply omit the h1 element and add the no-title class to the card element. This will adjust the padding and margin to ensure proper spacing.

<div class="card md">
    <h1>Card Title</h1>
    <p>This is an example of a basic card.</p>
</div>

Card with Buttons

You can include buttons inside a card by adding a container with the buttons class. This will properly space and align the buttons within the card.

<div class="card md">
    <h1>Card with Buttons</h1>
    <p>This card contains buttons.</p>
    <div class="buttons">
        <button class="button primary">OK</button>
        <button class="button secondary">Cancel</button>
    </div>
</div>

Card with Close Button

To add a close button to a card, include a container with the close class. This will position the container in the top-right corner of the card.

You can add any button style inside the close container, but it's common to use a square button with a close icon (×).

<div class="card md">
    <h1>Card with Close Button</h1>
    <p>This card has a close button.</p>
    <div class="close">
        <button class="button square danger">&times;</button>
    </div>
</div>

Title Alignment

By default, card titles are left-aligned. To center the title, add the center class to the h1 element. To right-align the title, use the right class.

<div class="card md">
    <h1 class="center">Centered</h1>
    <p>The title of this card is centered.</p>
</div>
