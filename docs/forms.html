Form Elements

Toon CSS offers styled form elements to create visually appealing and user-friendly forms. Below are examples of various form components.

Login

Usage

To use the form styles, simply use the form element along with the input class for input fields. Buttons within the form should use the button class.

The form layout is managed using CSS Grid to ensure proper alignment of labels and input fields. This layout helps maintain a clean and organized appearance. Simply place your labels and inputs within the form element as shown in the example below.

You can specify the size of the form by adding one of the following classes to the form element: xxs, xs, sm, md, lg, xl and xxl.

Form Titles

Use a h1 for the card title, and regular text elements for the content. h2 can be used for subheadings.

To create a form without a title, simply omit the h1 element and add the no-title class to the form element. This will adjust the padding and margin to ensure proper spacing.

Buttons in Forms

You can insert buttons into the form as needed, using a container with the buttons class to group them together, and properly align them within the form.

Text Inputs

Apply the input class to standard input fields to style them consistently with the Toon CSS theme. This includes all input types, except the image type. textarea elements also receive the same styling when inside a form.

<form>
    <h1>Text Input Example</h1>

    <label for="text-input">Text Input:</label>
    <input type="text" id="text-input" class="input" placeholder="Enter text" />

    <div class="buttons">
        <button type="reset" class="button secondary">Reset</button>
        <button type="submit" class="button primary">Submit</button>
    </div>
</form>

Textarea

Use the textarea element for multi-line text input. When placed inside a form, it will automatically receive the appropriate styling consistent with other input fields.

<form>
    <h1>Textarea Example</h1>

    <label for="message">Message:</label>
    <textarea id="message" rows="4" placeholder="Enter your message here..."></textarea>

    <div class="buttons">
        <button type="reset" class="button secondary">Reset</button>
        <button type="submit" class="button primary">Submit</button>
    </div>
</form>

Checkbox Example

Use the checkbox input type to create checkboxes within your form. Each checkbox should be paired with a corresponding label for clarity.

<form>
    <h1>Checkbox Example</h1>

    <label for="checkbox1">Option 1:</label>
    <input type="checkbox" id="checkbox1" />

    <label for="checkbox2">Option 2:</label>
    <input type="checkbox" id="checkbox2" checked />

    <div class="buttons">
        <button type="reset" class="button secondary">Reset</button>
        <button type="submit" class="button primary">Submit</button>
    </div>
</form>

Checkbox Example

Radio Buttons

Use the radio input type to create radio buttons within your form. Wrap radio buttons in a container with the radio class to ensure proper spacing and alignment within the group.

<form>
    <h1>Radio Button Example</h1>

    <div class="radio">
        <input type="radio" id="radio1" name="radio-group" checked />
        <label for="radio1">Radio Option 1</label>
        <input type="radio" id="radio2" name="radio-group" />
        <label for="radio2">Radio Option 2</label>
        <input type="radio" id="radio3" name="radio-group" />
        <label for="radio3">Radio Option 3</label>
    </div>

    <div class="buttons">
        <button type="reset" class="button secondary">Reset</button>
        <button type="submit" class="button primary">Submit</button>
    </div>
</form>

Select Dropdown

Use the select element to create dropdown menus within your form. Each select should be paired with a corresponding label for clarity.

<form>
    <h1>Select Dropdown Example</h1>

    <label for="select">Select:</label>
    <select id="select">
        <option value="" disabled selected>Select an option</option>
        <option value="option1">Option 1</option>
        <option value="option2">Option 2</option>
        <option value="option3">Option 3</option>
    </select>

    <div class="buttons">
        <button type="reset" class="button secondary">Reset</button>
        <button type="submit" class="button primary">Submit</button>
    </div>
</form>

Fieldset

Use the fieldset element to group related form controls together. The legend element provides a caption for the fieldset.

<form>
    <h1>Fieldset Example</h1>

    <fieldset>
        <legend>Fieldset</legend>

        <label for="email2">Email:</label>
        <input type="email" id="email2" placeholder="you@example.com" />

        <label for="password2">Password:</label>
        <input type="password" id="password2" placeholder="password" />
    </fieldset>

    <div class="buttons">
        <button type="reset" class="button secondary">Reset</button>
        <button type="submit" class="button primary">Submit</button>
    </div>
</form>
