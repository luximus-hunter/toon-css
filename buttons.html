Usage

Normal Buttons

To use the button style, simply add the button class to your element. You can also add a variant class to change the button's color and style. The default button style is secondary. For example, to create a primary button, use the following code:

<button class="button primary">Primary</button>

Supported elements include <button> and <a> tags

Disabled Buttons

To disable a button, simply add the disabled attribute to your button element. This will apply a disabled style to the button and prevent any interaction.

<button class="button warning" disabled>Warning</button>

Square Buttons

To create square buttons, add the square class alongside the button class. This will remove the border radius, giving the button a square appearance.

<button class="button red square">×</button>

Button Groups

To create a group of buttons, wrap your button elements inside a container with the group class. This will apply proper spacing and alignment to the buttons within the group.

<div class="group">
    <button class="button blue">Blue</button>
    <button class="button purple">Purple</button>
    <button class="button pink">Pink</button>
</div>
